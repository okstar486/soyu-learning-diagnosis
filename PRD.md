# 📋 소유 학습 진단 앱 - 제품 요구사항 정의서 (PRD)

## 1. 제품 개요

### 1.1 비전
학습에 어려움을 겪는 아이의 **진짜 원인**을 찾아주는 맞춤형 진단 도구

### 1.2 핵심 가설
```
"능력 부족"이 아니라 "학습에 대한 자기효능감 붕괴"
```

귀멸의칼날 74,000단어 소설을 읽은 아이가 학습 지문을 못 읽는 건 능력 문제가 아닙니다.

### 1.3 목표
1. **원인 파악**: 왜 힘든가? (능력 vs 동기 vs 정서 vs 학습스타일)
2. **현재 수준**: 어휘력, 읽기 이해 등 객관적 수준
3. **강점 발견**: 자신감 재료 확보
4. **학습 동기 평가**: 회피 패턴, 동기 유형
5. **맞춤 전략**: 부모용 구체적 개입 포인트

### 1.4 성공 지표

| 지표 | 목표 |
|------|------|
| 진단 완료율 | 80% 이상 (6개 영역 모두 완료) |
| 1회 참여 시간 | 15~30분 유지 |
| 재참여 의향 | "또 하고 싶다" 응답 |
| 부모 인사이트 | 구체적 개입 포인트 3개 이상 도출 |

## 2. 사용자 정의

### 2.1 Primary User: 소유
- **연령**: 초6 → 중1 (입학까지 약 2개월)
- **기기**: 갤럭시탭 14.6인치
- **특성**:
  - 귀멸의칼날 50부작 소설 완독 (74,000단어)
  - 게임 고도 몰입, 도전정신
  - 학습에 대한 자기효능감 붕괴
  - 쓰기 거부, 주관식 회피

### 2.2 Secondary User: 부모
- **역할**: 결과 확인, 개입 전략 수립
- **접근**: 비밀번호 보호 대시보드

### 3.2 진단 영역 상세

#### 영역 1: 어휘력
- **목적**: 전과목 학습 병목 확인
- **문항 수**: 20-30개
- **유형**: 동의어/반의어, 빈칸 채우기, 문맥 추론
- **난이도**: 초4 ~ 중1

#### 영역 2: 학습 자기효능감
- **목적**: 근본 원인 파악
- **문항 수**: 15-20개
- **유형**: 리커트 척도, 시나리오 선택
- **측정**: 자기 인식, 회복 탄력성, 도전 성향

#### 영역 3: 학습맥락 읽기 이해
- **목적**: 지문 이해력 파악
- **문항 수**: 10-15개
- **유형**: 짧은 지문 + 선택형
- **측정**: 사실적/추론적 이해, 구조 파악

#### 영역 4: 학습 동기/정서
- **목적**: 회피 패턴 분석
- **문항 수**: 15-20개
- **유형**: 상황별 감정/행동 선택
- **측정**: 내적/외적 동기, 학습 불안, 귀인 양식

#### 영역 5: 문법 기초
- **목적**: 국어 기초 점검
- **문항 수**: 15-20개
- **유형**: 틀린 부분 찾기, 올바른 표현 선택
- **측정**: 품사, 문장 성분, 맞춤법

#### 영역 6: 강점 발견
- **목적**: 자신감 재료 확보
- **문항 수**: 10-15개
- **유형**: 선호도 선택, 체크리스트
- **측정**: 학습 스타일, 강점 영역

## 5. UX 원칙

### 5.1 설계 원칙
1. **쓰기 최소화**: 모든 입력은 선택형 또는 타이핑
2. **즉각적 피드백**: 정답/오답 즉시 하루 반응
3. **실패 없음**: "다시 해보자!" (오답 ≠ 실패)
4. **시각적 재미**: 애니메이션, 이펙트, 캐릭터
5. **진행 명확**: 진행률 바, 남은 문제 수

### 5.2 테마: 귀멸의칼날 + 하루
- **컨셉**: 나비저택 특별 수련
- **진단**: 호흡 수련
- **문제**: 훈련 미션
- **정답**: 기술 성공
- **오답**: 다시 수련 (실패 아님)

### 5.3 색상 팔레트
```css
--primary: #9370DB;      /* 보라 (하루) */
--secondary: #E6E6FA;    /* 연보라 */
--accent: #FFD700;       /* 금색 (레벨업) */
--bg-main: #1a1a2e;      /* 어두운 남색 */
--success: #4ade80;      /* 정답 - 녹색 */
--retry: #fbbf24;        /* 재시도 - 노란색 */
```
