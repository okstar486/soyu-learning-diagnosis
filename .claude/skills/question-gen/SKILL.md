# 진단 문제 생성

## JSON 구조

### 전체 파일 구조
```json
{
  "area": "vocabulary",
  "areaName": "어휘력",
  "areaNameKorean": "어휘력",
  "description": "전과목 학습의 병목인 어휘력 수준을 파악합니다",
  "themeStory": "혈귀의 이름을 외워야 해!",
  "icon": "📚",
  "totalQuestions": 20,
  "estimatedTime": "10분",
  "questions": [...]
}
```

### 개별 문제 구조
```json
{
  "id": "vocab_001",
  "type": "choice",
  "difficulty": 3,
  "question": "문제 텍스트",
  "context": "하루가 나비저택에서...",
  "options": ["A", "B", "C", "D"],
  "answer": 0,
  "hint": "힌트 텍스트 (선택)",
  "explanation": "정답 설명 (선택)",
  "feedback": {
    "correct": "잘했어! 하루도 기뻐해!",
    "incorrect": "다시 한번 생각해볼까?"
  },
  "theme": {
    "character": "하루",
    "situation": "훈련 중"
  }
}
```

## 문제 유형 (type)
- `choice`: 4지선다 선택형
- `scale`: 5점 리커트 척도
- `scenario`: 시나리오 기반 선택
- `ranking`: 순위 매기기

## 난이도 기준 (difficulty)
| 레벨 | 대상 | 설명 |
|------|------|------|
| 1 | 초4 수준 | 기초 어휘, 단순 문장 |
| 2 | 초5 수준 | 일상 어휘, 복합 문장 |
| 3 | 초6 수준 | 교과 어휘, 추론 포함 |
| 4 | 중1 기초 | 학술 어휘, 논리 구조 |
| 5 | 중1 심화 | 고급 어휘, 복합 추론 |

## 영역별 문항 수
| 영역 | 최소 | 권장 |
|------|------|------|
| 어휘력 | 20 | 25-30 |
| 자기효능감 | 15 | 18-20 |
| 읽기 이해 | 10 | 12-15 |
| 동기/정서 | 15 | 18-20 |
| 문법 기초 | 15 | 18-20 |
| 강점 발견 | 10 | 12-15 |

## 테마 적용 가이드

### 귀멸의칼날 캐릭터 활용
- 탄지로: 성실함, 노력
- 젠이츠: 두려움 극복, 잠재력
- 이노스케: 도전정신
- 시노부: 지식, 가르침
- 토키토: 집중력, 재능

### 하루 캐릭터 활용
- 여우의 호흡 수련
- 나비저택 훈련
- 동료와 함께 성장
- 구미호 가면의 힘

## 피드백 작성 원칙

### 정답 피드백 (correct)
- 긍정적이고 구체적인 칭찬
- 하루 캐릭터의 기쁨 표현
- 다음 문제로의 동기 부여

**예시**:
- "잘했어! 하루도 너와 함께라 든든해!"
- "역시! 여우의 호흡이 느껴져!"
- "대단해! 이 조자로 가면 형을 익힐 수 있어!"

### 오답 피드백 (incorrect)
- "실패" 단어 절대 사용 금지
- 재도전 유도
- 긍정적 격려

**예시**:
- "다시 해보자! 하루도 기다리고 있어!"
- "괜찮아, 한 번 더 생각해볼까?"
- "수련은 반복이야. 다시 도전해보자!"
