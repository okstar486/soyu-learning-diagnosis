## 3. 난이도 기준

### 3.1 난이도 레벨

| 레벨 | 대상 | 어휘 | 문장 복잡도 | 추론 요구 |
|------|------|------|-------------|-----------|
| 1 | 초4 | 기초 어휘 | 단문 | 없음 |
| 2 | 초5 | 일상 어휘 | 복문 | 약간 |
| 3 | 초6 | 교과 어휘 | 복합문 | 보통 |
| 4 | 중1 기초 | 학술 어휘 | 논리 구조 | 높음 |
| 5 | 중1 심화 | 고급 어휘 | 복합 논리 | 매우 높음 |

### 3.2 난이도 조절 알고리즘

```javascript
// 연속 정답 시 난이도 상승
if (consecutiveCorrect >= 3) {
  difficulty = Math.min(difficulty + 1, 5);
}

// 연속 오답 시 난이도 하락
if (consecutiveWrong >= 2) {
  difficulty = Math.max(difficulty - 1, 1);
}

// 목표: 70-80% 정답률 유지
```

## 4. 영역별 문제 가이드

### 4.1 어휘력 (vocabulary)

#### 문제 유형
- 동의어/반의어 찾기
- 빈칸에 적절한 단어
- 문맥에서 의미 추론
- 관용구/속담 이해

#### 테마 적용
```
예문: "탄지로는 ___한 의지로 훈련을 계속했다."
     → 귀멸 캐릭터와 상황 활용
```

#### 문항 수: 20-30개

### 4.2 자기효능감 (self-efficacy)

#### 문제 유형
- 리커트 척도 (5점)
- 시나리오 선택
- 자기 인식 질문

#### 측정 요소
- 학습 능력 자기 인식
- 실패 후 회복 탄력성
- 도전 vs 회피 성향
- 목표 달성 기대

#### 문항 수: 15-20개

### 4.3 읽기 이해 (reading)

#### 문제 유형
- 짧은 지문 + 선택형
- 중심 내용 찾기
- 빈칸 추론
- 순서 배열

#### 지문 특성
- 200-300자 (난이도별 조절)
- 귀멸 세계관 배경
- 친숙한 상황 설정

#### 문항 수: 10-15개

### 4.4 동기/정서 (motivation)

#### 문제 유형
- 상황별 감정 선택
- 행동 선택 시나리오
- 선호도 순위

#### 측정 요소
- 내적/외적 동기
- 학습 불안 수준
- 회피 트리거
- 귀인 양식

#### 문항 수: 15-20개

### 4.5 문법 기초 (grammar)

#### 문제 유형
- 품사 구분
- 문장 성분 찾기
- 맞춤법 선택
- 올바른 표현 선택

#### 출제 범위
- 품사 (명사, 동사, 형용사 등)
- 문장 성분 (주어, 서술어, 목적어)
- 맞춤법/띄어쓰기
- 높임 표현

#### 문항 수: 15-20개

### 4.6 강점 발견 (strength)

#### 문제 유형
- 선호도 선택
- 강점 체크리스트
- 성공 경험 회상

#### 측정 요소
- 학습 스타일 (시각/청각/체험)
- 강점 영역 (언어/논리/예술/관계)
- 흥미 분야
- 성공 경험

#### 문항 수: 10-15개

## 5. 피드백 작성 가이드

### 5.1 정답 피드백 원칙
1. 구체적 칭찬
2. 하루 캐릭터 감정 표현
3. 다음 문제로의 동기 부여

**좋은 예**:
```
"정확해! 하루도 기뻐하고 있어!"
"대단해! 이 기세로 다음 문제도 가자!"
"역시! 네 실력이 느껴져!"
```

### 5.2 오답 피드백 원칙
1. "실패", "틀렸어" 금지
2. 재도전 유도
3. 힌트 제공
4. 긍정적 격려

**좋은 예**:
```
"다시 생각해볼까? 힌트: ..."
"괜찮아! 한 번 더 도전해보자!"
"아깝다! 조금만 더 생각해봐!"
```

**나쁜 예** (금지):
```
"틀렸어"
"실패했어"
"못 맞췄네"
```
