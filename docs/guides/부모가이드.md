## 1. 앱 소개 및 목적

### 1.1 이 앱을 만든 이유

**"왜 우리 아이는 좋아하는 소설은 잘 읽으면서 학습 지문은 회피할까?"**

이 앱은 학습 어려움의 **진짜 원인**을 찾기 위해 만들어졌습니다.

```
핵심 가설: "능력 부족"이 아니라 "학습에 대한 자기효능감 붕괴"
```

귀멸의 칼날 소설 74,000단어를 읽은 아이가 학습 지문을 못 읽는 건 **능력 문제가 아닙니다**.
문제는 "나는 공부를 못한다"는 믿음, 즉 **자기효능감의 붕괴**입니다.

### 1.2 앱의 목표

| 목표 | 설명 |
|------|------|
| **원인 파악** | 왜 힘들어할까? (능력 vs 동기 vs 정서 vs 학습스타일) |
| **현재 수준 확인** | 어휘력, 읽기 이해 등 객관적 수준 측정 |
| **강점 발견** | 자신감 회복을 위한 재료 확보 |
| **동기 평가** | 회피 패턴과 동기 유형 분석 |
| **맞춤 전략 수립** | 부모가 실행할 수 있는 구체적 개입 포인트 제공 |

### 1.3 이 앱의 특별한 점

#### 1.3.1 귀멸의 칼날 + 하루 캐릭터 테마
- 아이가 좋아하는 콘텐츠로 동기 부여
- 하루 캐릭터가 함께 수련하며 격려
- "진단"이 아닌 "나비저택 특별 수련"

#### 1.3.2 쓰기 최소화 설계
- 모든 문제는 선택형 또는 타이핑
- 손으로 쓰는 부담 제거
- 연필 잡기 어려움에 대한 배려

#### 1.3.3 실패 없는 설계
- "틀렸어" 대신 "다시 해보자!"
- 격려 중심 피드백
- 레벨업으로 성취감 제공

#### 1.3.4 게이미피케이션
- 여우의 호흡 7형 레벨 시스템
- 정답마다 경험치 획득
- 레벨업 연출과 축하

## 2. 시작하기

### 2.1 설치 방법

#### 방법 1: 웹 브라우저에서 바로 사용
1. 웹 브라우저(크롬, 사파리 등)에서 접속
   ```
   https://okstar486.github.io/soyu-learning-diagnosis
   ```
2. 바로 사용 가능

#### 방법 2: 홈 화면에 앱 추가 (권장)
**안드로이드 (갤럭시탭):**
1. 크롬 브라우저에서 앱 접속
2. 오른쪽 상단 메뉴(⋮) → "홈 화면에 추가"
3. "설치" 또는 "추가" 선택
4. 홈 화면에 앱 아이콘 생성 완료

**iOS (아이패드):**
1. 사파리에서 앱 접속
2. 하단 공유 버튼(□↑) → "홈 화면에 추가"
3. "추가" 선택
4. 홈 화면에 앱 아이콘 생성 완료

### 2.2 첫 실행

1. **로딩 화면**
   - "나비저택에 도착 중..." 표시
   - 하루 캐릭터 등장

2. **환영 메시지**
   - 하루: "안녕! 나는 하루야. 오늘도 함께 수련하자!"

3. **홈 화면**
   - 6개 영역 선택 가능
   - 진행 상황 표시
   - 현재 레벨 표시 (여우의 호흡 1형부터 시작)

### 2.3 데이터 저장

#### 자동 저장
- 진행 상황은 자동으로 저장됩니다
- 언제든지 중단하고 나중에 이어서 할 수 있습니다
- 브라우저의 localStorage에 저장 (기기에만 저장, 클라우드 X)

#### 데이터 백업 (권장)
1. 부모 대시보드 접속
2. "결과 다운로드" 버튼 클릭
3. JSON 파일로 저장
4. 중요한 결과는 백업 필수

## 3. 진단 진행 방법

### 3.1 진단 영역 (총 6개)

| 순서 | 영역 | 목적 | 예상 시간 |
|------|------|------|-----------|
| 1 | **어휘력** | 전과목 학습의 병목 확인 | 10-15분 |
| 2 | **학습 자기효능감** | 근본 원인 파악 | 10-15분 |
| 3 | **읽기 이해** | 지문 이해력 측정 | 10-15분 |
| 4 | **학습 동기/정서** | 회피 패턴 분석 | 10-15분 |
| 5 | **문법 기초** | 기초 수준 점검 | 10-15분 |
| 6 | **강점 발견** | 자신감 재료 확보 | 10-15분 |

**총 예상 시간**: 60-90분 (여러 번에 나누어 진행 가능)

### 3.2 진단 진행 팁

#### 3.2.1 한 번에 다 하지 않기
```
✅ 권장: 하루 1-2개 영역 (15-30분)
❌ 비권장: 한 번에 전체 (피로도 높음)
```

#### 3.2.2 아이의 컨디션 고려
- 식사 직후는 피하기
- 피곤하거나 짜증난 상태에서는 다음에
- 게임이나 좋아하는 활동 후 보상으로 연결 가능

#### 3.2.3 환경 조성
- 조용한 공간
- 편안한 자세
- 충분한 화면 밝기
- 방해 요소 제거 (알림 끄기)

#### 3.2.4 부모의 역할
```
✅ DO: 격려, 관심, 칭찬
❌ DON'T: 지켜보기, 정답 알려주기, 재촉하기
```

**가장 좋은 방법**: 아이 혼자 진행하도록 하고, 나중에 대시보드로 결과 확인

### 3.3 진단 중 아이가 보는 것

#### 아이 화면 구성
- **상단**: 현재 레벨 (여우의 호흡 X형)
- **중앙**: 문제 및 선택지
- **하단**: 진행률 바 (X / 20)
- **캐릭터**: 하루가 옆에서 함께 수련

#### 피드백 방식
- **정답**: "잘했어! 역시 너야!" + 경험치 획득
- **오답**: "다시 해보자! 하루도 기다릴게!" + 재도전 유도
- **영역 완료**: 레벨업 연출 + "새로운 형 습득!"

#### 아이가 받는 결과
- 점수나 백분율 표시 **없음**
- 레벨 형태로만 표시 ("여우의 호흡 3형!")
- 강점 중심 피드백

## 4. 부모 대시보드 사용법

### 4.1 접속 방법

1. **홈 화면**에서 하단 "부모 대시보드" 버튼 클릭
2. **비밀번호 입력**: `7942`
3. 대시보드 진입

> **비밀번호 관리**: 아이가 보지 않도록 주의하세요. 점수나 분석 결과를 보면 자신감에 영향을 줄 수 있습니다.

### 4.2 대시보드 구성

#### 4.2.1 전체 요약
```
┌─────────────────────────────────────┐
│  진단 진행률: 83% (5/6 완료)        │
│  현재 레벨: 여우의 호흡 5형         │
│  총 경험치: 450 exp                 │
└─────────────────────────────────────┘
```

#### 4.2.2 레이더 차트
6개 영역을 한눈에 볼 수 있는 육각형 차트
- 각 축: 영역별 점수 (0-100)
- 강점과 약점 시각적으로 확인

#### 4.2.3 영역별 상세 결과

**어휘력**
```
점수: 72/100 (중간)
정답률: 18/25 (72%)
난이도 분포: 쉬움(5) 보통(10) 어려움(3)
강점: 일상어, 관심사 관련 어휘
약점: 추상어, 학습 전문 용어
```

**자기효능감**
```
점수: 45/100 (낮음) ⚠️
분석: 학습에 대한 자신감 매우 낮음
패턴: "나는 못한다" 응답 多
권장: 작은 성공 경험 누적 필요
```

(각 영역별 동일한 형식)

#### 4.2.4 종합 분석

**강점 영역**
- 관심사 관련 이해력 (귀멸의칼날 등)
- 게임 내 문제 해결력
- 동료와의 협력 성향

**주의 필요 영역**
- 학습 자기효능감 ⚠️
- 학습 동기 (외적 동기 의존)
- 문법 기초

**개입 우선순위**
1. 자기효능감 회복 (가장 중요)
2. 성공 경험 누적
3. 어휘력 확장

### 4.3 기능

#### 결과 다운로드
- JSON 파일로 저장
- 백업 및 기록 용도
- 전문가 상담 시 제출 가능

#### 비교 보기 (향후 기능)
- 이전 진단 결과와 비교
- 성장 추이 확인

#### 인쇄/공유
- 브라우저 인쇄 기능 사용
- PDF로 저장 가능

## 5. 결과 확인 및 해석

### 5.1 점수 의미

#### 점수 범위
| 점수 | 해석 | 의미 |
|------|------|------|
| **80-100** | 우수 | 이 영역은 강점! 칭찬과 격려 |
| **60-79** | 중간 | 평균 수준, 조금만 더 노력 |
| **40-59** | 낮음 | 개입 필요, 체계적 지원 |
| **0-39** | 매우 낮음 | 즉시 개입, 전문가 상담 고려 |

> **중요**: 점수는 아이에게 절대 보여주지 마세요. 수치가 아닌 강점과 가능성에 집중하세요.

### 5.2 영역별 해석 요약

**어휘력**
- **높음**: 독서량 많음, 배경지식 풍부
- **낮음**: 독서 활동 필요, 어휘 확장 전략

**자기효능감**
- **높음**: 학습에 자신감 있음
- **낮음**: ⚠️ 가장 중요한 개입 포인트, 성공 경험 누적 필요

**읽기 이해**
- **높음**: 지문 이해력 좋음
- **낮음**: 독해 전략 훈련 필요

**동기/정서**
- **높음**: 내적 동기 있음
- **낮음**: 외적 보상 의존, 동기 부여 전략 필요

**문법**
- **높음**: 기초 탄탄
- **낮음**: 체계적 학습 필요

**강점 발견**
- 아이의 학습 스타일과 강점 영역 확인
- 이것을 활용한 학습 전략 수립

### 5.3 주의 신호 (이럴 때 전문가 상담)

```
⚠️ 즉시 전문가 상담이 필요한 경우:

1. 자기효능감 점수 30 이하
2. 모든 영역 점수 40 이하
3. 진단 중 극심한 거부감 표시
4. "나는 바보야", "못해" 등 부정적 자기 인식 반복
5. 학습 관련 불안, 공포 증상
6. 신체 증상 (두통, 복통, 손떨림 등)
```

**상담 기관**
- 학교 상담교사
- Wee센터
- 아동청소년상담센터
- 학습클리닉 (병원)

## 6. 자주 묻는 질문

### 6.1 사용 관련

**Q1. 진단은 몇 번 해야 하나요?**
```
A. 첫 진단 후 2-3개월마다 재진단을 권장합니다.
   개입 전략을 실행한 후 변화를 확인하는 용도로 활용하세요.
```

**Q2. 아이가 중간에 그만두려고 해요.**
```
A. 괜찮습니다. 억지로 하지 마세요.
   - 저장되므로 나중에 이어서 가능
   - 1-2개 영역만 해도 충분히 의미 있음
   - 보상과 연결 ("수련 끝나면 로블록스 30분")
```

**Q3. 점수가 너무 낮게 나왔어요.**
```
A. 점수는 현재 상태일 뿐, 아이의 가치가 아닙니다.
   - 낮은 점수는 도움이 필요한 영역을 알려주는 것
   - 개입 전략매뉴얼을 참고하여 실행
   - 변화에는 시간이 걸립니다 (최소 2-3개월)
```

**Q4. 아이가 대충 찍는 것 같아요.**
```
A. 다음을 확인하세요:
   - 피곤하거나 짜증난 상태인가?
   - 동기 부여가 부족한가?
   - 너무 어렵거나 지루한가?

   → 다른 시간에 다시 시도
   → 보상 연결
   → 함께 옆에서 응원 (정답 말하지 말기)
```

### 6.2 기술 관련

**Q5. 오프라인에서도 되나요?**
```
A. 네! PWA로 설치하면 오프라인에서도 작동합니다.
   단, 최초 1회는 인터넷 연결 필요.
```

**Q6. 데이터가 어디에 저장되나요?**
```
A. 아이가 사용하는 기기(탭)의 브라우저에만 저장됩니다.
   - 클라우드에 저장 안 됨 (개인정보 보호)
   - 다른 기기에서는 볼 수 없음
   - 브라우저 데이터 삭제 시 사라짐

   → 중요한 결과는 JSON 다운로드로 백업!
```

**Q7. 다른 아이도 사용할 수 있나요?**
```
A. 이 앱은 "소유"를 위해 맞춤 제작되었습니다.
   다른 아이에게는 맞지 않을 수 있습니다.

   단, 유사한 상황의 아이라면 참고용으로 활용 가능:
   - 초6~중1
   - 귀멸의칼날 좋아함
   - 학습 자기효능감 낮음
   - 쓰기 어려움
```

### 6.3 결과 해석 관련

**Q8. 아이에게 결과를 어떻게 말해줘야 하나요?**
```
A. 점수는 절대 말하지 마세요. 강점 중심으로 전달하세요.

✅ 좋은 예:
   "하루랑 수련 재밌었어? 너 이번에 5형까지 올랐대! 대단해!"
   "엄마가 봤는데 너 [강점 영역]이 엄청 좋더라! 자랑스러워!"

❌ 나쁜 예:
   "자기효능감이 45점이네? 왜 이렇게 낮아?"
   "어휘력이 약하다는데 책 더 읽어야겠다."
```

**Q9. 점수가 높은데도 학교 성적은 안 좋아요.**
```
A. 진단은 "잠재력"을 보는 것입니다.
   실제 성적에는 다른 요인이 영향을 줍니다:
   - 시험 불안
   - 시간 관리
   - 쓰기 능력
   - 학습 습관

   → 잠재력은 있으니 환경과 지원을 조정하세요.
```

**Q10. 전문가 상담은 언제 받아야 하나요?**
```
A. 다음 경우 전문가 상담을 권장합니다:
   - 자기효능감 30 이하
   - 여러 영역 40 이하
   - 진단 거부 또는 극심한 스트레스 반응
   - 3개월 개입 후에도 변화 없음

   전문가는 적(敵)이 아니라 동반자입니다.
```

### 6.4 활용 관련

**Q11. 이 결과로 무엇을 해야 하나요?**
```
A. 다음 순서로 진행하세요:

1. 결과 확인 (부모 대시보드)
2. 강점과 약점 파악
3. 개입전략매뉴얼 읽기
4. 우선순위 결정 (보통 자기효능감 1순위)
5. 1-2개 전략 선택하여 실행
6. 2-3개월 후 재진단
```

**Q12. 학교 선생님께 보여줘도 되나요?**
```
A. 네, 좋습니다!
   - JSON 파일 또는 화면 캡처
   - 담임, 상담교사와 공유
   - 협력적 지원 체계 구축

   단, 아이가 모르게 하세요. (낙인 방지)
```

## 7. 부모님께 드리는 말

### 7.1 가장 중요한 것

```
이 앱은 "아이를 평가하는 도구"가 아닙니다.
"아이를 이해하고 도울 방법을 찾는 도구"입니다.
```

### 7.2 기억해주세요

1. **점수가 아이의 가치가 아닙니다**
   - 낮은 점수 = 도움이 필요한 영역
   - 높은 점수 = 강점으로 활용할 영역

2. **변화에는 시간이 걸립니다**
   - 최소 2-3개월
   - 급하게 밀어붙이지 마세요
   - 작은 변화를 축하하세요

3. **부모의 믿음이 가장 큽니다**
   - "넌 할 수 있어" > 백 마디 잔소리
   - 점수 상승 < 자신감 회복

4. **혼자 하지 마세요**
   - 학교, 전문가와 협력
   - 부모 커뮤니티 활용
   - 필요하면 도움 요청

### 7.3 응원 메시지

```
아이를 위해 이 앱을 찾아주신 당신은
이미 훌륭한 부모입니다.

아이는 소중하고 가능성으로 가득 차 있습니다.

함께 천천히, 따뜻하게 걸어가세요.
하루처럼, 넘어져도 다시 일어서며.

- 소유 학습 진단 앱 제작팀
```
