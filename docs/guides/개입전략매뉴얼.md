## 1. 개입 전략 기본 원칙

### 1.1 시작하기 전에

```
⚠️ 중요한 마음가짐:

1. 급하게 하지 마세요 (최소 2-3개월)
2. 한 번에 1-2가지만 (욕심 금물)
3. 작은 변화를 축하하세요
4. 아이를 믿으세요
5. 부모도 쉬어가세요
```

### 1.2 우선순위 결정

#### Step 1: 긴급도 확인

```
자기효능감 < 40  →  최우선 개입
신체/정서 증상   →  전문가 상담 병행
기초 능력 부족   →  체계적 지원
```

#### Step 2: 1-2개 전략 선택

```
❌ 6개 영역 동시 개입
✅ 가장 낮은 1-2개 영역 집중
```

#### Step 3: 실행 가능한 것부터

```
이상적인 전략 < 실행 가능한 전략
```

### 1.3 효과 측정

#### 관찰 포인트
- [ ] 아이의 표정이 밝아졌나?
- [ ] 학습 회피가 줄었나?
- [ ] "못해" 말이 줄었나?
- [ ] 스스로 시도하나?
- [ ] 부모-자녀 관계가 나아졌나?

#### 재진단 시점
- 2-3개월 후 재진단
- 변화 추이 확인
- 전략 조정

## 2. 영역별 개입 전략

### 2.1 어휘력 (Vocabulary)

#### 목표
```
학습에 필요한 어휘를 확장하고
새로운 단어를 스스로 익히는 습관 형성
```

#### 전략 1: 일상 대화에서 어휘 확장

**방법:**
```
하루 3개 새 단어 목표

예시 대화:
아이: "엄마, 배고파"
부모: "출출하구나? '출출하다'는 조금 배고프다는 뜻이야"
아이: "응, 출출해"
부모: "좋아! 새 단어 하나 배웠네!"

→ 강요 없이, 자연스럽게
→ 사용하면 칭찬
→ 아이가 좋아하는 맥락에서
```

**실행 팁:**
- 관심사(귀멸의칼날 등) 관련 어휘부터
- 부모도 같이 배우는 척
- 어휘 노트 만들기 (선택)
- 게임처럼: "오늘의 단어 퀴즈!"

#### 전략 2: 독서 활동

**방법:**
```
매일 30분 독서 (강제 X)

선택 기준:
1. 아이가 좋아하는 장르 (귀멸의칼날 등)
2. 난이도: 80% 이해 가능한 수준
3. 두께: 얇은 책부터

함께 읽기:
- 부모가 읽어주기 (초기)
- 번갈아 읽기 (중기)
- 혼자 읽기 + 얘기 나누기 (후기)
```

**실행 팁:**
- 로블록스 전 15분 독서 (보상 연결)
- 도서관 함께 가기 (주 1회)
- 읽은 책 리스트 만들기
- 강제 독후감 X

#### 전략 3: 어휘 게임/앱

**추천 활동:**
```
1. 끝말잇기 (가족 게임)
2. 스무고개 (어휘 설명 연습)
3. 십자말풀이
4. 단어 카드 게임
5. 어휘 앱 (게임형)
```

**실행 팁:**
- 가족 게임 시간에 자연스럽게
- 이기면 보상 (스티커 등)
- 부모도 진지하게 참여
- 재미 우선, 학습은 부차적

#### 전략 4: 교과서 어휘 미리 익히기

**방법:**
```
다음 주 학습 단원 미리 보기
→ 어려운 단어 5개 선택
→ 그림/예문으로 설명
→ 일상 대화에서 사용

예: "광합성" 배울 예정
→ 식물 키우며 "광합성" 설명
→ "햇빛으로 밥 먹는 거야"
→ 아이가 말로 설명하게
```

#### 점검 체크리스트

- [ ] 주 3회 이상 새 단어 노출
- [ ] 매일 30분 독서 (또는 주 5일)
- [ ] 주 1회 어휘 게임
- [ ] 아이가 새 단어 사용 시 칭찬
- [ ] 한 달에 책 2-3권 읽기

### 2.2 학습 자기효능감 (Self-Efficacy)

#### 목표
```
"나는 할 수 있다"는 믿음 회복
작은 성공 경험 누적
```

> **가장 중요한 영역!** 자기효능감이 낮다면 다른 모든 전략보다 이것을 우선하세요.

#### 전략 1: 1일 1성공 경험

**방법:**
```
매일 100% 성공 가능한 과제 제공

예시:
- 아침: "침대 정리" → 성공 → 칭찬
- 오후: "책상 정리" → 성공 → 칭찬
- 저녁: "양치질" → 성공 → 칭찬

학습 관련:
- 문제 1개 풀기 (쉬운 것)
- 단어 3개 외우기
- 5분 책 읽기

→ 작지만 확실한 성공
→ 매일 반복
→ 한 달 = 30번 성공
```

**실행 팁:**
- 절대 실패하지 않을 수준으로
- 성공 시 즉각 칭찬
- 성공 스티커/도장 (시각화)
- 일주일에 한 번 "이번 주 성공 리스트" 리뷰

#### 전략 2: 과정 칭찬 (결과 아님)

**잘못된 칭찬:**
```
❌ "100점 맞았네! 대단해!"
❌ "1등 했어? 최고야!"
❌ "이번엔 잘했네"
```

**올바른 칭찬:**
```
✅ "30분 동안 집중했네! 대단해!"
✅ "모르는 거 물어봤구나! 용기 있어!"
✅ "어려워도 끝까지 했네! 멋져!"
✅ "다시 도전했구나! 성장하고 있어!"
```

**차이점:**
- 결과 칭찬: 통제 불가 (운, 실력)
- 과정 칭찬: 통제 가능 (노력, 태도)

→ 과정 칭찬이 자기효능감을 높입니다!

#### 전략 3: 실패를 학습 기회로

**실패 시 대화:**
```
아이: "틀렸어... 나 바보야..."

❌ 나쁜 반응:
   "왜 이것도 못해?"
   "집중 안 했지?"
   "다시 해봐"

✅ 좋은 반응:
   "아깝다! 거의 다 왔는데!"
   "어디서 막혔어? 함께 보자"
   "한 번 시도한 거 자체가 대단해"
   "실수는 배우는 과정이야"

→ 실패 = 나쁜 것 (X)
→ 실패 = 배움의 기회 (O)
```

#### 전략 4: 자기 대화 바꾸기

**부정적 자기 대화:**
```
"나는 못해"
"나는 바보야"
"어차피 안 될 거야"
```

**긍정적 자기 대화로 전환:**
```
부모: "나는 못해" 대신 뭐라고 하면 좋을까?
아이: "......"
부모: "나는 아직 못해. 하지만 배우고 있어" 어때?
아이: "나는 아직 못해..."
부모: "좋아! '아직'이 중요해!"

→ "못해" → "아직 못해"
→ "바보야" → "배우는 중이야"
→ "안 될 거야" → "해볼 거야"
```

#### 전략 5: 역할 모델 제시

**방법:**
```
1. 부모의 실패 경험 공유
   "아빠도 어릴 때 수학 못했어.
    하지만 포기 안 하고 연습했더니 지금은 잘해"

2. 아이가 좋아하는 인물
   "하루(캐릭터)도 처음엔 여우의 호흡 1형도 못했어.
    하지만 연습해서 9형까지 익혔잖아?"

3. 과거의 아이
   "너 작년에 [이것] 못했잖아?
    지금은 잘하고 있어. 너 성장했어!"
```

#### 점검 체크리스트

- [ ] 매일 1개 성공 경험 제공
- [ ] 과정 칭찬 하루 3번 이상
- [ ] "못해" 말 듣으면 "아직 못해"로 전환
- [ ] 실패 시 격려 대화
- [ ] 일주일에 1번 성공 리뷰

### 2.3 읽기 이해 (Reading Comprehension)

#### 목표
```
학습 지문을 이해하고
핵심을 파악하는 능력 향상
```

#### 전략 1: 함께 읽기

**방법:**
```
교과서/문제집 지문을 함께 읽기

Step 1: 부모가 읽어주기 (소리 내서)
Step 2: 아이에게 묻기
   "무슨 내용이었어?"
   "주인공이 뭐 했어?"
   "왜 그랬을까?"

Step 3: 다시 읽기 (아이가)
Step 4: 핵심 찾기
   "이 글에서 가장 중요한 건 뭘까?"

→ 이해 못 해도 괜찮음
→ 과정이 중요
```

**실행 팁:**
- 하루 10분 (짧게)
- 아이가 좋아하는 주제부터
- 그림/도표 함께 보기
- 핵심 문장 밑줄 긋기

#### 전략 2: 요약하기 연습

**방법:**
```
3문장 요약 훈련

긴 지문 읽은 후:
부모: "3문장으로 요약해볼까?"
아이: "으음..."
부모: "처음엔 뭐가 나왔어?"
아이: "OOO가 XXX했어요"
부모: "좋아! 그다음은?"

→ 처음엔 5문장도 괜찮음
→ 점점 3문장으로 줄이기
→ 핵심만 남기기 연습
```

**실행 팁:**
- 만화책부터 시작 (부담 적음)
- 하루 1개 지문
- 요약을 글이 아닌 말로
- 그림으로 표현도 OK

#### 전략 3: 질문 만들기

**방법:**
```
지문 읽고 질문 만들기

아이가 선생님이 되어:
"이 지문으로 문제를 낸다면?"

예:
지문: "식물은 광합성을 한다..."
아이 질문: "광합성이 뭐예요?"
부모: "좋은 질문! 답은?"
아이: "햇빛으로 양분 만드는 거요"
부모: "완벽!"

→ 질문 만들기 = 이해의 증거
```

#### 전략 4: 그림/도식화

**방법:**
```
지문을 그림으로 표현

특히 시각형 학습자에게 효과적

예: "삼각형의 넓이"
→ 실제 삼각형 그리기
→ 밑변, 높이 표시
→ 공식 옆에 그림

예: "역사 사건"
→ 타임라인 그리기
→ 인물 관계도

→ 이해 못 하면 그림 못 그림
→ 그림 그리면서 이해됨
```

#### 점검 체크리스트

- [ ] 주 5일, 10분 함께 읽기
- [ ] 읽은 내용 3문장 요약
- [ ] 질문 만들어보기 (주 3회)
- [ ] 그림/도표로 표현 (주 2회)
- [ ] 아이가 설명할 기회 주기

### 2.4 동기/정서 (Motivation & Emotion)

#### 목표
```
외적 동기 → 내적 동기 전환
학습 불안 감소
```

#### 전략 1: 보상 체계 설계 (외적 → 내적)

**단계별 전환:**
```
1단계: 외적 보상 (즉각)
   문제 10개 풀면 → 로블록스 30분

2단계: 외적 보상 (지연)
   일주일 매일 → 주말에 로블록스 2시간

3단계: 사회적 보상
   "잘했어!" 칭찬 → 아이가 기뻐함

4단계: 내적 보상
   "알았을 때 기분 좋아!" → 스스로 만족

→ 급하게 전환 X
→ 최소 6개월-1년
```

**현실적 보상 예시:**
```
즉각 보상: 스티커, 간식
단기 보상: 로블록스 시간, 용돈
장기 보상: 외출, 선물
사회적: 칭찬, 인정, 자랑
```

#### 전략 2: 선택권 주기

**방법:**
```
"해라" 대신 "골라봐"

❌ "국어 숙제 해"
✅ "국어 먼저 할래? 수학 먼저 할래?"

❌ "책 읽어"
✅ "이 책? 저 책? 네가 골라봐"

❌ "10분 공부해"
✅ "5분 할래? 10분 할래?"

→ 선택 = 통제감
→ 통제감 = 동기
```

#### 전략 3: 스트레스 관리

**학습 전 루틴:**
```
1. 심호흡 3회
2. "할 수 있어" 말하기
3. 타이머 설정 (부담 줄이기)
4. 시작!

→ 불안 감소
→ 집중력 향상
```

**학습 중 쉼표:**
```
25분 집중 → 5분 휴식 (포모도로)

휴식 시:
- 스트레칭
- 물 마시기
- 잠깐 걷기
- 좋아하는 노래 1곡

→ 번아웃 예방
```

#### 전략 4: 관심사 연결

**방법:**
```
아이가 좋아하는 것 → 학습 연결

예: 귀멸의칼날 좋아함
→ 국어: 귀멸 소설 읽기
→ 수학: "하루가 혈귀 9마리 처치, 토키토는..."
→ 역사: "다이쇼 시대 배경"
→ 영어: "Demon Slayer 영어 대사"

→ 흥미 = 동기
```

#### 점검 체크리스트

- [ ] 보상 체계 운영 (일관성)
- [ ] 하루 3번 선택권 주기
- [ ] 학습 전 루틴 (심호흡 등)
- [ ] 25-5 휴식 패턴
- [ ] 관심사 연결 시도 (주 1회)

### 2.5 문법 기초 (Grammar)

#### 목표
```
품사, 문장 성분, 맞춤법 등
국어 문법 기초 다지기
```

#### 전략 1: 교정 피드백 (긍정적)

**방법:**
```
틀린 표현 → 자연스럽게 교정

아이: "이거 재밌어요!"
부모: "정말 재미있구나!" (있다/없다 교정)

아이: "밥 먹고 싶어요"
부모: "밥 먹고 싶구나!" (먹다+고 싶다)

→ "틀렸어!" 지적 X
→ 올바른 표현 모델링
→ 자연스럽게 흡수
```

**주의:**
- 한 번에 1개만 교정
- 아이가 재사용하면 칭찬
- 잘못 쓴다고 비난 X

#### 전략 2: 문법 게임

**추천 게임:**
```
1. 품사 찾기 게임
   광고판, 간판에서 명사/동사 찾기

2. 문장 만들기
   주사위 3개: 주어, 서술어, 목적어
   → 문장 완성

3. 맞춤법 퀴즈
   "되/돼", "안 돼/안돼" 골라봐

→ 게임 = 재미
→ 재미 = 학습
```

#### 전략 3: 책 읽으며 문법

**방법:**
```
책 읽다가 문법 포인트 발견

"여기 '~는'이 나왔네? 이건 주격 조사야"
"이 문장, 주어가 뭘까?"

→ 강의 X
→ 발견의 기쁨
→ 자연스럽게 노출
```

#### 전략 4: 일기 쓰기 (선택)

**방법:**
```
쓰기 부담 있으면 X
가능하면 주 1회 짧은 일기

일기 후 함께 보기:
- 맞춤법 1-2개만 교정
- 좋은 표현 칭찬
- 내용 공감

→ 완벽함 추구 X
→ 쓰는 습관 형성
```

#### 점검 체크리스트

- [ ] 틀린 표현 자연스럽게 교정 (하루 1회)
- [ ] 문법 게임 (주 2회)
- [ ] 책 읽으며 문법 발견 (주 3회)
- [ ] 일기 또는 짧은 글 (주 1회, 선택)

---

### 2.6 강점 발견 (Strengths)

#### 목표
```
아이의 강점을 찾아
학습에 활용하기
```

#### 전략 1: 강점 기반 학습

**학습 스타일별:**

**시각형 (그림, 차트)**
```
- 마인드맵으로 정리
- 그림/도표 활용
- 색깔 펜으로 필기
- 영상 자료 활용
```

**청각형 (소리, 리듬)**
```
- 소리 내서 읽기
- 노래로 외우기
- 설명 듣고 학습
- 토론/대화식 학습
```

**체험형 (몸으로, 만지며)**
```
- 실험/만들기
- 역할극
- 움직이며 암기
- 프로젝트 학습
```

#### 전략 2: 강점 칭찬

**방법:**
```
"너는 [강점]이 정말 뛰어나!"
"이 [강점]으로 [학습]도 할 수 있어!"

예:
- 몰입력 → "게임처럼 공부도 몰입해볼까?"
- 스토리텔링 → "이야기로 만들어볼까?"
- 관계 → "친구랑 같이 공부하면?"
```

#### 전략 3: 강점 확장

**방법:**
```
강점을 더 강화 → 자신감 → 약점 개선

예:
그림 잘 그림
→ 그림 수업 (자신감)
→ 그림으로 개념 정리 (학습)
→ "난 이것도 잘해!" (자기효능감)
→ 다른 것도 도전
```

#### 점검 체크리스트

- [ ] 아이의 강점 3가지 파악
- [ ] 강점 활용 학습 시도 (주 2회)
- [ ] 강점 칭찬 (매일)
- [ ] 강점 확장 활동 (월 1회)

## 3. 일일/주간 활동 제안

### 3.1 하루 루틴 예시 (평일)

```
06:30  기상
       ✅ "침대 정리" → 첫 성공 경험

07:00  아침 식사
       ✅ 오늘의 단어 1개 (자연스럽게)

15:00  귀가 후 휴식
       - 간식, 게임 30분

15:30  학습 시간 1 (25분)
       ✅ 학교 숙제 (함께, 필요 시)
       ✅ 과정 칭찬 "집중 잘했어!"

16:00  휴식 (5분)

16:05  학습 시간 2 (25분)
       ✅ 독서 또는 어휘 게임
       ✅ 선택권 "뭐부터 할래?"

16:30  자유 시간
       - 로블록스 또는 자유 활동

18:00  저녁 식사
       ✅ 오늘의 성공 나누기
       "오늘 뭐 잘했어?"

21:00  취침 준비
       ✅ "양치질" → 마지막 성공 경험
       ✅ 칭찬과 함께 "오늘도 잘했어!"
```

### 3.2 주간 계획 예시

```
월: 어휘력 (독서 30분 + 어휘 3개)
화: 자기효능감 (성공 경험 + 칭찬)
수: 읽기 이해 (함께 읽기 + 요약)
목: 동기/정서 (보상 활동 + 선택권)
금: 문법 (게임 + 자연스런 교정)
토: 강점 활동 (관심사 연계 활동)
일: 휴식 (가족 활동, 자유 시간)
```

### 3.3 월간 점검

```
매월 마지막 주 일요일:

1. 이번 달 성공 리스트 리뷰
   "이것들을 해냈구나!"

2. 다음 달 목표 1-2개
   "다음 달엔 이것도 해볼까?"

3. 부모 스스로 점검
   - 칭찬 충분히 했나?
   - 과정에 집중했나?
   - 아이와의 관계는?
   - 나도 쉬었나?

4. 조정
   - 너무 많으면 줄이기
   - 잘 되면 유지
   - 안 되면 다른 전략
```

## 4. 부모-자녀 대화 팁

### 4.1 상황별 대화 스크립트

#### 상황 1: 숙제 거부

**❌ 피해야 할 대화:**
```
부모: "숙제 안 해?"
아이: "싫어..."
부모: "해야지! 안 하면 혼날 거야!"
아이: (울음, 짜증)
```

**✅ 권장 대화:**
```
부모: "숙제가 힘들어 보이네?"
아이: "응... 어려워..."
부모: "어떤 게 제일 어려워?"
아이: "이 문제..."
부모: "그럼 이것만 함께 볼까? 5분만"
아이: "응..."
부모: (함께 풂) "다 했네! 대단해!"
```

#### 상황 2: "나는 못해" 발언

**❌ 피해야 할 대화:**
```
아이: "나는 못해..."
부모: "왜 못해? 할 수 있어!"
아이: "난 바보야..."
부모: "그런 말 하지 마!"
```

**✅ 권장 대화:**
```
아이: "나는 못해..."
부모: "못한다고 느껴지는구나. 속상하겠다"
아이: "응..."
부모: "너 작년에 [이것] 못했는데 지금은 잘하잖아?"
아이: "그랬나?"
부모: "응! 그때도 '못해'라고 했는데, 지금은 해.
       지금 이것도 곧 하게 될 거야"
아이: "...그럴까?"
부모: "엄마/아빠가 믿어. 그리고 함께 할 거야"
```

#### 상황 3: 시험 망쳤을 때

**❌ 피해야 할 대화:**
```
부모: "점수가 왜 이래?"
아이: "......"
부모: "공부 안 했지?"
아이: (울음)
부모: "다음엔 잘해야지!"
```

**✅ 권장 대화:**
```
부모: "시험 어땠어?"
아이: "...못 봤어..."
부모: "많이 속상하겠다. 엄마/아빠한테 얘기해줘"
아이: (얘기)
부모: (공감) "그랬구나. 그런데 말이야,
       엄마/아빠는 점수보다 네가 노력한 거 알아.
       매일 조금씩 한 거 다 봤어"
아이: "근데 점수가..."
부모: "점수는 지금 상태일 뿐이야.
       계속 노력하면 언젠가 올라가.
       중요한 건 포기 안 하는 거야"
```

### 4.2 칭찬 대화법

#### 구체적 칭찬

```
❌ "잘했어"
✅ "30분 동안 집중해서 문제 푼 거 대단해!"

❌ "똑똑하네"
✅ "모르는 거 물어본 거 용기 있어!"

❌ "착하네"
✅ "동생 도와준 거 참 따뜻하다!"
```

#### 과정 칭찬

```
✅ "시도한 것 자체가 대단해"
✅ "틀려도 다시 도전한 거 멋져"
✅ "어려워도 끝까지 한 거 최고야"
✅ "스스로 한 거 자랑스러워"
```

### 4.3 경청 기술

```
1. 눈 맞추기
   "말해줘. 엄마/아빠가 듣고 있어"

2. 반복하기
   아이: "선생님이 혼냈어"
   부모: "선생님이 혼내셔서 속상했구나"

3. 공감하기
   "그랬으면 많이 힘들었겠다"
   "네가 그렇게 느낄 수 있어"

4. 판단 보류
   "그래서?" (계속 듣기)
   "더 얘기해봐"

5. 해결 강요 X
   "어떻게 하면 좋을까?" (아이에게 물어보기)
```

### 4.4 피해야 할 말

```
❌ "왜 이것도 못해?"
❌ "친구는 잘하던데?"
❌ "그래서 엄마가 뭐라고 했어?"
❌ "내가 너 때는..."
❌ "공부 안 하면 나중에..."
❌ "게임만 하면 돼?"
❌ "하기 싫으면 하지 마"
❌ "엄마/아빠 속 썩이지 마"
```

### 4.5 권장하는 말

```
✅ "힘들었겠다"
✅ "엄마/아빠가 도와줄게"
✅ "천천히 해도 괜찮아"
✅ "넌 할 수 있어"
✅ "함께 하자"
✅ "네 마음이 궁금해"
✅ "노력하는 거 알아"
✅ "네가 자랑스러워"
```

## 5. 전문가 연계 안내

### 5.1 협력 체계 구축

```
    [아이]
      |
  ----+----
  |   |   |
부모 학교 전문가

→ 3자 협력이 가장 효과적
```

### 5.2 학교와의 협력

#### 담임 선생님과의 대화

**준비 사항:**
```
1. 진단 결과 (간단 요약)
2. 아이의 특성
3. 부모가 하고 있는 노력
4. 학교에 부탁하고 싶은 것
```

**대화 예시:**
```
"선생님, 시간 내주셔서 감사합니다.
저희 아이가 학습에 어려움이 있어서 진단을 해봤는데요,
특히 자기효능감이 낮게 나왔습니다.

집에서는 [이런 노력]을 하고 있는데,
학교에서도 [이렇게] 해주시면 감사하겠습니다.

예를 들어, 작은 성공 경험을 주시거나,
과정을 칭찬해주시면 아이가 자신감을 얻을 것 같습니다.

협력해서 함께 도와주시면 좋겠습니다."
```

#### 상담 교사 활용

```
- 정기 상담 (월 1회)
- 또래 관계 문제
- 학습 동기 프로그램
- 진로 탐색
```

### 5.3 전문 기관 연계

#### Wee센터

**대상:**
```
- 학습 어려움
- 또래 관계 문제
- 정서/행동 문제
- 가정 문제
```

**서비스:**
```
- 1:1 상담 (무료)
- 집단 상담
- 학습 전략 프로그램
- 심리 검사
```

**신청:**
```
학교 상담교사 통해 의뢰
또는 직접 전화 (지역별 Wee센터)
```

#### 학습클리닉 (병원)

**대상:**
```
- ADHD 의심
- 학습 장애 의심
- 심한 불안/우울
- 약물 치료 필요
```

**서비스:**
```
- 의료적 진단
- 약물 치료 (필요 시)
- 인지 치료
- 부모 교육
```

**비용:**
```
보험 적용 가능 (진료비의 일부)
```

### 5.4 부모 지원 그룹

#### 온라인 커뮤니티
```
- 맘카페 (학습 어려움 게시판)
- 네이버 카페 (ADHD, 학습 장애 등)
- 페이스북 그룹
```

#### 오프라인 모임
```
- 지역 Wee센터 부모 교육
- 도서관 부모 강좌
- 학교 학부모 모임
```

**효과:**
```
- 정서적 지지
- 정보 교환
- 고립감 해소
- 실패담 공유 (위안)
```

### 5.5 부모 스스로 돌보기

```
⚠️ 중요:

아이를 돕기 위해 부모가 번아웃되면 안 됩니다!
```

#### 부모 셀프 체크

- [ ] 나도 잘 쉬고 있나?
- [ ] 배우자/가족과 나누고 있나?
- [ ] 너무 죄책감 느끼지 않나?
- [ ] 나만의 시간이 있나?
- [ ] 도움 요청할 사람이 있나?

#### 부모를 위한 조언

```
1. 완벽한 부모는 없습니다
   "충분히 좋은 부모(Good Enough Parent)"로 충분

2. 작은 변화를 축하하세요
   아이뿐 아니라 내 노력도

3. 혼자 하지 마세요
   배우자, 가족, 친구, 전문가와 함께

4. 장기전입니다
   마라톤처럼, 페이스 조절

5. 나도 쉬어야 합니다
   쉬는 것도 전략입니다
```

## 6. 마무리

### 6.1 핵심 요약

```
1. 우선순위: 자기효능감 > 다른 영역
2. 원칙: 작게, 천천히, 꾸준히
3. 방법: 칭찬, 성공 경험, 강점 활용
4. 협력: 학교, 전문가와 함께
5. 셀프케어: 부모도 쉬기
```

### 6.2 실행 체크리스트

#### 이번 주 시작할 것 (1-2개만)

- [ ] 매일 1성공 경험 제공
- [ ] 과정 칭찬 하루 3번
- [ ] 함께 책 읽기 10분
- [ ] 어휘 3개 노출
- [ ] 선택권 주기
- [ ] _______ (직접 작성)

#### 한 달 후 점검

- [ ] 아이의 변화 관찰
- [ ] 효과 있는 전략 계속
- [ ] 안 되는 전략 조정
- [ ] 재진단 고려

### 6.3 응원 메시지

```
당신은 이미 훌륭한 부모입니다.
이 매뉴얼을 읽었다는 것만으로도요.

완벽하게 하지 못해도 괜찮습니다.
아이는 완벽한 부모가 아니라
노력하는 부모를 보며 배웁니다.

함께 천천히, 따뜻하게 걸어가세요.

아이는 반드시 자랄 것입니다.
그리고 당신의 사랑을 기억할 것입니다.
```
