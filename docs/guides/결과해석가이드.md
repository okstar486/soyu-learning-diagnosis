## 1. 6개 영역별 의미

### 1.1 어휘력 (Vocabulary)

#### 무엇을 측정하나요?

```
"학습의 기초 체력"을 측정합니다.
어휘력은 모든 과목 학습의 병목입니다.
```

| 측정 요소 | 설명 | 예시 |
|----------|------|------|
| **어휘 폭** | 알고 있는 단어 수 | 일상어, 학습어, 전문어 |
| **어휘 깊이** | 단어의 정확한 의미 이해 | 다의어, 유의어, 반의어 |
| **문맥 이해** | 상황에 맞는 어휘 사용 | "달리다"의 여러 의미 |

#### 점수별 해석

| 점수 | 상태 | 의미 | 조치 |
|------|------|------|------|
| **80-100** | 우수 | 독서량 많음, 배경지식 풍부 | 계속 독서 + 심화 어휘 |
| **60-79** | 중간 | 기본은 되지만 확장 필요 | 독서 + 어휘 게임 |
| **40-59** | 낮음 | 학습에 방해되는 수준 | 체계적 어휘 학습 필요 |
| **0-39** | 매우 낮음 | 즉시 개입 필요 | 전문가 상담 + 집중 지도 |

#### 구체적 신호

**높을 때 (80+)**
- 책을 많이 읽음
- 대화 시 적절한 어휘 사용
- 새로운 단어를 스스로 익힘

**낮을 때 (40-)**
- "무슨 말인지 모르겠어" 자주 말함
- 교과서 지문 이해 못함
- 수학 문제도 어휘 때문에 못 풀음

#### 특별 고려사항 (소유의 경우)

```
⚠️ 중요:
소유는 귀멸의칼날 74,000단어를 읽었습니다.
→ 능력은 있다는 증거!

만약 어휘력이 낮다면:
→ "관심 없는 영역" 어휘 부족
→ "학습 맥락" 어휘 회피
→ 동기 문제이지 능력 문제 아님
```

### 1.2 학습 자기효능감 (Self-Efficacy)

#### 무엇을 측정하나요?

```
"나는 할 수 있다"는 믿음을 측정합니다.
이것이 이 앱의 핵심 진단 영역입니다.
```

| 측정 요소 | 설명 | 예시 질문 |
|----------|------|-----------|
| **자기 인식** | 학습 능력에 대한 믿음 | "나는 공부를 잘할 수 있다" |
| **회복 탄력성** | 실패 후 다시 시도 의지 | "틀려도 다시 도전한다" |
| **도전 성향** | 어려운 과제 앞에서 태도 | "어려워도 시도한다" vs "피한다" |
| **목표 설정** | 학습 목표 설정 및 기대 | "노력하면 성적이 오를 것이다" |

#### 점수별 해석

| 점수 | 상태 | 의미 | 심각도 |
|------|------|------|--------|
| **80-100** | 우수 | 학습에 자신감 있음, 도전적 | 💚 건강 |
| **60-79** | 중간 | 불안정하지만 기본은 됨 | 💛 관찰 필요 |
| **40-59** | 낮음 | 자기효능감 붕괴 시작 | 🧡 즉시 개입 |
| **0-39** | 매우 낮음 | 학습된 무기력 상태 | ❤️ 긴급 개입 |

#### 구체적 신호

**높을 때 (80+)**
- "해볼게요!" 태도
- 틀려도 다시 도전
- "연습하면 잘할 수 있어요"
- 목표 설정하고 도전

**낮을 때 (40-)**
- "나는 못해요" 자주 말함
- 시도 전에 포기
- "해도 안 될 거예요"
- 모르면 바로 찍기/빈칸
- 회피, 거짓말, 대충 마무리

#### 핵심 진단 포인트

```
⚠️ 가장 중요한 영역!

게임:  "나는 깰 수 있다" → 도전 → 성공 → 더 도전
공부:  "나는 못한다" → 회피 → 실패 → 더 회피

이것이 학습된 무기력(Learned Helplessness)입니다.

→ 점수가 낮다면: 최우선 개입 영역
→ 점수가 높다면: 다른 문제는 해결 가능
```

#### 부모 체크리스트

자기효능감이 낮은 아이는 이런 말을 자주 합니다:
- [ ] "나는 바보야"
- [ ] "어차피 못할 거야"
- [ ] "선생님이 싫어해"
- [ ] "친구들은 다 잘하는데 나만..."
- [ ] "공부는 재미없어"
- [ ] "안 해도 돼요? 귀찮아..."

✅ 3개 이상: 자기효능감 문제 가능성 높음

### 1.3 학습맥락 읽기 이해 (Reading Comprehension)

#### 무엇을 측정하나요?

```
"학습 상황에서" 지문을 이해하는 능력을 측정합니다.
(재미있는 소설이 아닌 교과서, 문제집 지문)
```

| 측정 요소 | 설명 | 예시 |
|----------|------|------|
| **사실적 이해** | 지문에 명시된 정보 파악 | 누가, 언제, 어디서, 무엇을 |
| **추론적 이해** | 암시된 정보 추론 | 왜, 어떻게, 의도는 |
| **구조 파악** | 글의 흐름과 논리 | 서론-본론-결론 |

#### 점수별 해석

| 점수 | 상태 | 의미 | 조치 |
|------|------|------|------|
| **80-100** | 우수 | 지문 이해력 뛰어남 | 심화 독서 |
| **60-79** | 중간 | 기본 이해는 됨 | 독해 전략 훈련 |
| **40-59** | 낮음 | 지문 이해 어려움 | 함께 읽기 + 요약 연습 |
| **0-39** | 매우 낮음 | 글을 읽어도 의미 파악 못함 | 전문가 상담 |

#### 중요한 구분

```
읽기 능력 ≠ 읽기 이해

아이가 글자를 읽을 수 있어도 (읽기 능력)
의미를 이해하지 못할 수 있습니다 (읽기 이해)
```

#### 특별 고려사항 (소유의 경우)

| 맥락 | 읽기 이해 |
|------|----------|
| 귀멸의칼날 소설 | ⭕ 완벽 이해 + 암기 |
| 학습 지문 (교과서) | ❌ 회피 또는 이해 못함 |

**→ 차이의 원인:**
- 동기 (재미 vs 강제)
- 자기효능감 (할 수 있다 vs 못한다)
- 정서 (즐거움 vs 불안)

**→ 능력은 있다! 환경과 동기를 조정하면 해결 가능!**

### 1.4 학습 동기/정서 (Motivation & Emotion)

#### 무엇을 측정하나요?

```
"왜 공부하는가?" 그리고 "공부할 때 어떤 기분인가?"를 측정합니다.
```

| 측정 요소 | 설명 | 예시 |
|----------|------|------|
| **내적 동기** | 재미, 흥미, 호기심 | "알고 싶어서", "재밌어서" |
| **외적 동기** | 보상, 칭찬, 처벌 회피 | "로블록스 때문에", "혼날까봐" |
| **학습 불안** | 공부할 때 스트레스 수준 | 두통, 복통, 회피 |
| **회피 패턴** | 언제, 왜 회피하는가 | 쓰기, 긴 지문, 어려운 문제 |

#### 점수별 해석

| 점수 | 상태 | 의미 | 동기 유형 |
|------|------|------|-----------|
| **80-100** | 우수 | 내적 동기 높음 | "재밌어서 해요!" |
| **60-79** | 중간 | 내적+외적 혼합 | "칭찬받고 싶어요" |
| **40-59** | 낮음 | 외적 동기 의존 | "로블록스 때문에" |
| **0-39** | 매우 낮음 | 무동기, 무기력 | "의미 없어요" |

#### 내적 vs 외적 동기

```
내적 동기 (바람직):
"수학이 재밌어요"
"역사가 궁금해요"
"알면 뿌듯해요"

외적 동기 (단기 효과):
"게임 시간 받으려고"
"엄마가 혼내서"
"친구보다 잘하고 싶어서"

→ 외적 동기로 시작해도 괜찮습니다!
   점점 내적 동기로 전환하는 게 목표입니다.
```

#### 정서 신호

**건강한 정서**
- 공부 후 성취감
- 어려워도 즐거움
- 실수해도 괜찮음

**문제 정서** ⚠️
- 공부 시작 전부터 짜증
- 신체 증상 (두통, 복통)
- 거짓말, 회피
- 눈물, 공황

### 1.5 문법 기초 (Grammar)

#### 무엇을 측정하나요?

```
국어 문법의 기초 이해를 측정합니다.
(품사, 문장 성분, 맞춤법 등)
```

| 측정 요소 | 설명 | 예시 |
|----------|------|------|
| **품사** | 명사, 동사, 형용사 등 구분 | "달리다"는 동사 |
| **문장 성분** | 주어, 서술어, 목적어 등 | "나는 밥을 먹는다" |
| **맞춤법/띄어쓰기** | 올바른 표기법 | "되/돼", "안 돼/안돼" |
| **높임 표현** | 존댓말, 겸양어 | "드시다", "주무시다" |

#### 점수별 해석

| 점수 | 상태 | 의미 | 조치 |
|------|------|------|------|
| **80-100** | 우수 | 문법 기초 탄탄 | 심화 학습 |
| **60-79** | 중간 | 기본은 알지만 실수 多 | 연습 필요 |
| **40-59** | 낮음 | 기초 부족 | 체계적 학습 |
| **0-39** | 매우 낮음 | 문법 개념 부재 | 기초부터 다시 |

#### 중요성

```
문법은 "덜 중요"하지 않습니다!

문법이 약하면:
- 독해 오류 증가
- 쓰기 불가능
- 국어 성적 하락
- 영어 문법도 어려움
```

### 1.6 강점 발견 (Strengths)

#### 무엇을 측정하나요?

```
"이 아이의 특별함"을 찾습니다.
자신감 회복을 위한 재료를 확보합니다.
```

| 측정 요소 | 설명 | 예시 |
|----------|------|------|
| **학습 스타일** | 어떻게 배우길 좋아하나 | 시각, 청각, 체험 |
| **다중지능** | 어떤 영역이 강한가 | 언어, 논리, 예술, 관계 |
| **흥미 분야** | 무엇에 몰입하나 | 게임, 소설, 미술, 음악 |
| **성공 경험** | 언제 잘했나 | 과거 성취 회상 |

#### 결과 형태

점수가 아닌 **프로필** 형태로 제공:

```
강점 프로필:

학습 스타일: 시각형 (이미지, 차트로 배우면 잘함)
다중지능: 언어 지능, 관계 지능
흥미 분야: 스토리텔링, 게임, 친구 관계
특별한 재능: 몰입력, 암기력

추천 학습 방법:
- 만화/그림으로 개념 정리
- 스토리로 엮어서 학습
- 친구와 함께 공부
- 게임처럼 미션 설정
```

#### 활용 방법

```
강점을 찾았다면:

1. 아이에게 알려주기
   "너는 이런 게 정말 잘하더라!"

2. 학습에 연결하기
   "이 방법으로 공부하면 어때?"

3. 자신감 재료로 사용
   "이것도 잘하잖아, 공부도 할 수 있어!"
```

## 2. 점수 해석 방법

### 2.1 절대 점수 vs 상대적 패턴

```
중요: 점수 숫자보다 "패턴"을 보세요!
```

#### 예시 1: 전체적으로 낮지만 균형

```
어휘력:     45
자기효능감: 42
읽기 이해:  48
동기/정서:  40
문법:       38
강점 발견:  -

→ 해석: 전반적으로 낮지만 특정 병목은 없음
→ 조치: 전체적 학습 지원 + 자기효능감 우선
```

#### 예시 2: 능력은 있는데 자기효능감만 낮음

```
어휘력:     78
자기효능감: 35 ⚠️
읽기 이해:  72
동기/정서:  42
문법:       68
강점 발견:  -

→ 해석: 능력은 있는데 자신감 붕괴!
→ 조치: 자기효능감 회복이 최우선! (소유의 경우)
```

#### 예시 3: 자기효능감은 높은데 능력 부족

```
어휘력:     38
자기효능감: 82
읽기 이해:  42
동기/정서:  75
문법:       40
강점 발견:  -

→ 해석: 의지는 있는데 기초 부족
→ 조치: 체계적 학습 지원 (잘 따라올 것)
```

### 2.2 병목 찾기

```
가장 낮은 영역 = 병목
→ 다른 영역의 발현을 막는 요인

예: 자기효능감 30 + 어휘력 70
→ 어휘력은 있는데 자신감이 없어서 활용 못함
→ 자기효능감 올리면 어휘력도 발현됨!
```

### 2.3 점수 변화 추적

첫 진단 후 2-3개월마다 재진단하여 변화 추적:

| 영역 | 1차 (1월) | 2차 (4월) | 변화 | 해석 |
|------|-----------|-----------|------|------|
| 자기효능감 | 35 | 58 | +23 | 개입 효과! |
| 어휘력 | 78 | 82 | +4 | 유지 |
| 동기/정서 | 42 | 65 | +23 | 개선! |

## 3. 레이더 차트 읽는 법

### 3.1 레이더 차트란?

```
        어휘력
          /\
         /  \
    문법/    \자기효능감
       /      \
      /________\
 강점발견   읽기이해
      \        /
       \      /
    동기/정서
```

6개 영역을 육각형으로 표시한 차트

### 3.2 형태별 해석

#### 형태 1: 균형 잡힌 육각형

```
모든 영역이 비슷한 수준
→ 전체적으로 고르게 발달 또는 전체적으로 낮음
→ 특정 병목 없음
```

#### 형태 2: 찌그러진 모양

```
한쪽은 크고 한쪽은 작음
→ 명확한 강점과 약점
→ 약점 집중 개입 필요
```

#### 형태 3: 아래쪽이 움푹

```
어휘력, 문법은 괜찮은데
자기효능감, 동기가 낮음
→ 능력은 있는데 마음이 문제
→ 심리적 개입 우선
```

#### 형태 4: 위쪽이 움푹

```
자기효능감, 동기는 높은데
어휘력, 문법이 낮음
→ 의지는 있는데 기초 부족
→ 학습 지원 우선
```

### 3.3 면적 비교

```
큰 면적: 전체적으로 우수
작은 면적: 전체적으로 낮음

→ 면적보다 "형태(패턴)"가 더 중요!
```

## 4. 강점/약점 분석

### 4.1 강점 찾기

#### 강점 기준
- 점수 70 이상
- 다른 영역보다 20점 이상 높음
- 아이가 즐거워하는 활동과 연결

#### 강점 활용 전략

```
강점 → 약점 연결

예: 스토리텔링 강점 (귀멸의칼날)
→ 역사: 스토리처럼 설명
→ 과학: 실험을 스토리로
→ 수학: 문제에 스토리 입히기
```

#### 강점 칭찬 예시

```
"너 [강점]이 정말 대단하더라!"
"[강점] 있으면 [약점]도 충분히 할 수 있어!"
"[강점]을 이렇게 활용해볼까?"
```

### 4.2 약점 파악

#### 약점 기준
- 점수 50 이하
- 학습에 방해되는 수준
- 아이가 회피하는 영역

#### 약점 접근 방법

```
❌ 약점을 공격하지 마세요:
   "왜 이것도 못해?"
   "이거 좀 해라!"

✅ 약점을 이해하고 지원하세요:
   "이게 좀 어려웠구나"
   "어떻게 하면 쉬울까?"
   "엄마/아빠가 도와줄게"
```

### 4.3 우선순위 결정

#### 원칙: 자기효능감 1순위

```
자기효능감 < 50  →  무조건 1순위
자기효능감 ≥ 50  →  가장 낮은 영역부터
```

#### 예시

**케이스 1:**
- 자기효능감: 35 ⚠️
- 어휘력: 65
- 문법: 40

→ 우선순위: 자기효능감 > 문법 > 어휘력

**케이스 2:**
- 자기효능감: 68
- 어휘력: 42 ⚠️
- 문법: 38 ⚠️

→ 우선순위: 어휘력/문법 동시 진행

## 5. 주의 필요 신호

### 5.1 긴급 신호 (즉시 전문가 상담)

```
❤️ 긴급:

1. 자기효능감 30 이하
2. 모든 영역 40 이하
3. 진단 중 극심한 거부/공황
4. "죽고 싶어", "사라지고 싶어" 등 발언
5. 신체 증상: 구토, 실신, 과호흡
```

### 5.2 경고 신호 (관심 필요)

```
🧡 경고:

1. 자기효능감 40-50
2. 2개 이상 영역 40 이하
3. 진단 회피 또는 대충 찍기
4. "나는 바보야" 자주 말함
5. 학습 관련 불안 증가
```

### 5.3 관찰 신호 (지속 관찰)

```
💛 관찰:

1. 어떤 영역 60 전후로 불안정
2. 외적 동기 의존 (보상 없으면 안 함)
3. 특정 과목/활동 회피
4. 친구 관계에서 위축
```

### 5.4 신호별 조치

| 신호 | 조치 | 시급성 |
|------|------|--------|
| 긴급 | 즉시 전문가 상담 (1주 내) | 최우선 |
| 경고 | 개입 전략 즉시 실행 + 전문가 상담 고려 | 높음 |
| 관찰 | 개입 전략 실행 + 2-3개월 후 재진단 | 중간 |

## 6. 전문가 상담 시점

### 6.1 언제 전문가를 만나야 할까?

#### 즉시 상담 (1주 내)

```
1. 자기효능감 30 이하
2. 극심한 학습 불안/공황
3. 자해/자살 관련 발언
4. 신체 증상 심각 (구토, 실신 등)
5. 등교 거부
```

#### 빠른 상담 (1개월 내)

```
1. 자기효능감 40 이하
2. 여러 영역 40 이하
3. 3개월 개입 후에도 변화 없음
4. 부모-자녀 관계 극도 악화
5. 또래 관계 문제 심각
```

#### 상담 고려 (필요 시)

```
1. 자기효능감 50 전후
2. 특정 영역만 지속적으로 낮음
3. 개입 방법을 모르겠음
4. 부모가 지침 (번아웃)
```

### 6.2 어디로 가야 할까?

| 상담 기관 | 대상 | 비용 | 특징 |
|----------|------|------|------|
| **학교 상담교사** | 모든 학생 | 무료 | 접근성 좋음, 학교 상황 이해 |
| **Wee센터** | 초중고 | 무료 | 전문 상담사, 다양한 프로그램 |
| **청소년상담복지센터** | 9-24세 | 무료/저렴 | 지역별 운영 |
| **학습클리닉 (병원)** | 학습 장애 의심 | 보험 적용 | 의료적 진단 가능 |
| **사설 상담센터** | 맞춤 상담 | 유료 | 전문성 높음, 대기 짧음 |

### 6.3 상담 준비물

```
1. 진단 결과 (JSON 파일 또는 화면 캡처)
2. 학교 성적표 (최근 1-2학기)
3. 아이의 특성 메모
   - 좋아하는 것
   - 싫어하는 것
   - 언제부터 힘들어했나
   - 특별한 사건
4. 부모가 관찰한 내용
5. 질문 사항 정리
```

### 6.4 상담 시 전달 사항

```
"우리 아이가 학습에 어려움을 겪고 있어요.
진단 결과 [특히 이 영역]이 낮게 나왔습니다.

특히 이런 점이 걱정됩니다:
- [구체적 행동/발언]
- [변화된 모습]
- [부모가 시도한 것]

어떻게 도와줄 수 있을까요?"
```

### 6.5 전문가 상담이 두려우신가요?

```
Q. "상담 받으면 아이한테 낙인 찍히지 않나요?"
A. 아닙니다. 조기 개입이 낙인을 예방합니다.

Q. "나쁜 부모 같아서 부끄러워요."
A. 도움을 요청하는 것은 좋은 부모의 증거입니다.

Q. "상담으로 정말 나아질까요?"
A. 혼자보다 전문가와 함께할 때 효과가 큽니다.

Q. "비용이 부담돼요."
A. 무료 기관(학교, Wee센터)부터 시작하세요.
```

## 7. 종합 분석 예시

### 사례 1: 소유 (가상 결과)

```
[진단 결과]
어휘력:     68/100
자기효능감: 38/100 ⚠️
읽기 이해:  72/100
동기/정서:  45/100
문법:       58/100
강점:       시각적 학습, 스토리텔링, 몰입력

[해석]
- 능력은 충분히 있음 (읽기 이해 72)
- 자기효능감 붕괴가 핵심 문제 (38)
- 동기는 외적 의존 (45)
- 귀멸의칼날 74,000단어 독파 = 능력 증명
- 병목: "나는 못한다"는 믿음

[우선순위]
1순위: 자기효능감 회복
2순위: 동기 전환 (외적→내적)
3순위: 문법 기초 강화

[개입 전략]
- 작은 성공 경험 누적 (1일 1성취)
- 강점 활용 학습 (스토리텔링)
- 긍정 피드백 증대
- 과정 칭찬 (결과 아닌)
- 게임처럼 미션 설정

[전문가 상담]
자기효능감 38 → 1개월 내 상담 권장
```

### 사례 2: 능력 부족형

```
[진단 결과]
어휘력:     35/100 ⚠️
자기효능감: 75/100
읽기 이해:  38/100 ⚠️
동기/정서:  70/100
문법:       32/100 ⚠️
강점:       관계 지능, 체험 학습

[해석]
- 의지는 있는데 기초 부족
- 어휘력, 읽기, 문법 전반적으로 낮음
- 자기효능감 높음 = 잘 따라올 것
- 체계적 학습 필요

[우선순위]
1순위: 어휘력 확장 (병목)
2순위: 문법 기초
3순위: 읽기 전략

[개입 전략]
- 매일 독서 30분
- 어휘 게임/앱 활용
- 문법 기초 강의
- 친구와 함께 학습 (관계 지능 활용)
```

## 8. 부모님께

### 결과를 보는 올바른 자세

```
✅ DO:
- 아이의 현재 상태 이해하기
- 강점에 먼저 집중하기
- 패턴과 원인 파악하기
- 실행 가능한 전략 세우기
- 변화에 시간 주기

❌ DON'T:
- 점수로 아이 평가하기
- 약점만 보고 질책하기
- 다른 아이와 비교하기
- 당장 변화 기대하기
- 아이에게 결과 보여주기
```

### 마지막 당부

```
이 진단은 아이를 "평가"하는 게 아닙니다.
아이를 "이해"하고 "도움"을 주기 위한 것입니다.

점수가 낮다면:
→ 아이가 나쁜 게 아니라 도움이 필요한 것

점수가 높다면:
→ 아이의 노력을 인정하고 격려

가장 중요한 건:
부모의 믿음과 사랑입니다.

"넌 할 수 있어. 엄마/아빠가 믿어."
```
