> "나비저택의 새로운 가족: 하루의 성장일기" 공식 설정

## 평행 세계 선언

이 캐릭터는 **'귀멸의 칼날' 세계관의 평행 세계**에 존재합니다.
원작과 다른 설정이 있으며, 이는 의도된 창작입니다.

## 1. 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | 츠키카게 하루 (月影 春, Tsukikage Haru) |
| **이름 의미** | 月影(달그림자) + 春(봄) = "달빛 아래 피어나는 봄" |
| **나이** | 12세 → 16세 (시즌별 성장) |
| **생일** | 3월 21일 (춘분) |
| **호흡** | 여우의 호흡 (獸の呼吸 파생, 완전 창작) |
| **계급** | 계 → 병 → 정 → 갑 (시즌별 승급) |
| **소속** | 귀살대 나비저택 |
| **일륜도 색상** | 은빛 + 주황빛 (여우불 색) |

## 4. 성격

### 4.1 시즌별 성격 변화

| 시즌 | 핵심 특성 | 두려움 | 강점 |
|------|-----------|--------|------|
| **시즌 1** | 겁많음, 자책 | 혈귀, 실패, 상실 | 회피 재능, 관찰력 |
| **시즌 2** | 성장 중, 불안정 | 또 잃을까 두려움 | 끈기, 동료애 |
| **시즌 3** | 각성, 책임감 | 지키지 못할까 | 결단력, 희생정신 |
| **시즌 4** | 성숙, 사랑 | 없음 (극복) | 지혜, 따뜻함, 리더십 |

### 4.2 핵심 특성 (최종)

1. **따뜻함**: 동료를 챙기는 마음 (네즈코에게서 배움)
2. **끈기**: 포기하지 않는 도전정신 (렌고쿠에게서 배움)
3. **지혜**: 상황을 파악하는 통찰력 (여우에게서 배움)
4. **책임감**: 후배를 지키려는 마음 (시노부에게서 배움)
5. **사랑**: 증오가 아닌 사랑으로 싸움 (9형의 본질)

### 4.3 말투 특징

| 시즌 | 말투 | 예시 |
|------|------|------|
| 시즌 1 | 소심, 작은 목소리 | "저... 괜찮을까요...?" |
| 시즌 2 | 점점 밝아짐 | "해볼게요!" |
| 시즌 3 | 결의에 찬 | "반드시 지킬 거야!" |
| 시즌 4 | 따뜻하고 자신감 | "함께하면 할 수 있어!" |

### 4.4 말투 예시 (앱용 - 시즌 4 기준)

```
"안녕! 나는 하루야. 함께 수련하자!"
```

## 5. 여우의 호흡 완전 체계

### 5.1 호흡 개요

| 항목 | 내용 |
|------|------|
| **파생** | 수의 호흡 (獸の呼吸) 계열 |
| **창시자** | 츠키카게 하루 |
| **특징** | 속도, 회피, 지혜, 분신 |
| **시각 효과** | 은빛 + 주황빛 여우불 |
| **영적 연결** | 은빛 여우 (소타의 영혼) |

### 5.2 전체 9형 공식 체계

| 형 | 공식 명칭 | 한자 | 로마자 | 특성 | 습득 |
|----|----------|------|--------|------|------|
| **1형** | 여우불 섬광 | 狐火閃 | Kobi-Sen | 초고속 직선 돌진 베기 | 시즌1 7편 |
| **2형** | 구미 회전 | 九尾回転 | Kyubi-Kaiten | 회전하며 9연속 베기 | 시즌1 9편 |
| **3형** | 환영 보법 | 幻影歩法 | Genei-Hoho | 7개 잔상 생성 이동 | 시즌2 13편 |
| **4형** | 월광 난무 | 月光乱舞 | Gekko-Ranbu | 다중 연속 공격 | 시즌2 20편 |
| **5형** | 구미 연격 | 九尾連撃 | Kyubi-Rengeki | 9연속 강타 | 시즌3 26편 |
| **6형** | 여우비 | 狐雨 | Ko-U | 광역 다방향 공격 | 시즌3 28편 |
| **7형** | 구미 연진 | 九尾連陣 | Kyubi-Renjin | 팀 전술 연계 | 시즌3 30편 |
| **8형** | 여우의 구현 | 狐之具現 | Ko-no-Gugen | 5체 분신 동시 공격 | 시즌3 33편 |
| **9형** | 구미천호 | 九尾天狐 | Kyubi-Tenko | 사랑의 완전체, 최종오의 | 시즌3 35-37편 |

### 5.3 9형 특별 설명

| 항목 | 내용 |
|------|------|
| **본질** | 기술이 아닌 **마음** (사랑) |
| **발현 조건** | 지키고 싶은 사람을 향한 진심 |
| **시각 효과** | 9개의 여우꼬리 + 천호(天狐) 형상 |
| **위력** | 무잔급 혈귀 대응 가능 |
| **깨달음** | "검술의 극의는 기술이 아니라 마음" (렌고쿠) |

### 5.4 앱 레벨업 대사 (형별)

```yaml
1형 → 2형: "축하해! 여우의 호흡 2형, 구미 회전을 익혔어! 이제 시작이야!"
2형 → 3형: "대단해! 3형 환영 보법! 여우처럼 빠르게 움직일 수 있어!"
3형 → 4형: "우와! 4형 월광 난무! 달빛 아래 춤추는 여우야!"
4형 → 5형: "5형 구미 연격! 아홉 번의 일격! 정말 강해졌어!"
5형 → 6형: "6형 여우비! 비처럼 쏟아지는 공격! 멋져!"
6형 → 7형: "7형 구미 연진! 이제 팀과 함께 싸울 수 있어!"
7형 → 8형: "8형 여우의 구현! 분신술이라니! 대단해!"
8형 → 9형: "9형 구미천호! 사랑의 힘으로 완성! 너는 진정한 여우의 검사야!"
```

## 7. 감정 표현

### 7.1 기쁨 (정답, 레벨업)
- 눈이 반달 모양
- 밝은 미소
- 뛰어오르는 포즈
- 여우귀가 쫑긋

### 7.2 격려 (오답, 재시도)
- 부드러운 눈빛
- 주먹 쥔 파이팅 포즈
- 살짝 기운 고개
- "할 수 있어" 입 모양

### 7.3 집중 (문제 풀이 중)
- 진지한 표정
- 똑바로 응시
- 차분한 자세
- 여우 가면 반쯤 쓴 상태

### 7.4 축하 (레벨업, 완료)
- 눈에서 빛나는 효과
- 두 팔 벌린 환영 포즈
- 은빛 + 주황빛 오라
- 여우불 이펙트

### 7.5 슬픔 (회상, 실수)
- 눈물 맺힌 눈
- 고개 숙임
- 손 꼭 쥠
- 하지만 포기 안 하는 표정

### 7.6 결의 (각오, 도전)
- 날카로운 눈빛
- 칼 쥔 손
- 여우 가면 착용
- 여우불 발현
