# 🗡️ 귀멸의칼날 하루 스토리 설정

> 진단 앱과 귀멸의칼날 세계관 연결

## 1. 세계관 설정

### 1.1 시점
- **시기**: 무한열차편 이후, 유곽편 이전
- **상황**: 평화로운 훈련 기간
- **장소**: 나비저택 (蝶屋敷)

### 1.2 배경 설명
```
무잔과의 최종 전투를 앞두고,
귀살대원들은 나비저택에서 특별 훈련을 받고 있다.
하루도 동료들과 함께 여우의 호흡을 연마하며
더 강해지기 위한 수련을 계속하고 있다.
```

## 2. 진단 = 수련 컨셉

### 2.1 핵심 은유
| 진단 용어 | 귀멸 용어 | 설명 |
|-----------|-----------|------|
| 진단 | 특별 수련 | 나비저택 훈련 프로그램 |
| 문제 | 훈련 미션 | 형(型)을 익히기 위한 과제 |
| 정답 | 기술 성공 | 호흡 기술 발동 |
| 오답 | 다시 수련 | 실패가 아닌 연습 과정 |
| 레벨업 | 형 습득 | 여우의 호흡 새 형태 마스터 |
| 완료 | 졸업 | 훈련 프로그램 마스터 |

### 2.2 진행 스토리
```
[시작]
하루: "안녕! 나비저택에 온 걸 환영해!
      시노부 선생님이 특별 수련 프로그램을 만들었어.
      나와 함께 여우의 호흡을 익혀보자!"

[영역 진행]
하루: "이번 훈련은 [영역명]이야!
      집중해서 미션을 수행해봐!"

[정답 시]
하루: "잘했어! 기술 발동 성공이야!"

[오답 시]
하루: "괜찮아! 수련은 반복이 중요해.
      다시 한번 도전해보자!"

[레벨업]
하루: "축하해! 여우의 호흡 [N]형을 습득했어!
      점점 강해지고 있어!"

[완료]
하루: "모든 수련을 마쳤어! 정말 대단해!
      이제 너도 진정한 검사야!"
```

## 3. 영역별 스토리 연결

### 3.1 어휘력 - "혈귀 도감 훈련"
```yaml
상황: "혈귀를 물리치려면 그들의 이름과 특성을 알아야 해!"
등장 캐릭터: 시노부 (가르침), 탄지로 (응원)
배경: 나비저택 도서관
훈련 목표: 다양한 단어와 개념 익히기

시작 대사: "시노부 선생님이 혈귀 도감을 주셨어!
          단어의 뜻을 정확히 알면 혈귀를 물리칠 수 있대!"

완료 대사: "어휘 훈련 완료! 이제 어떤 혈귀가 나타나도 알아볼 수 있어!"
```

### 3.2 자기효능감 - "마음의 불꽃 훈련"
```yaml
상황: "진정한 힘은 마음에서 나와!"
등장 캐릭터: 렌고쿠 (영감), 하루 (동료)
배경: 나비저택 정원
훈련 목표: 자신감과 도전정신 확인

시작 대사: "렌고쿠 님이 말씀하셨어.
          '마음을 불태워라!' 네 마음의 불꽃은 어떤 색일까?"

완료 대사: "마음 훈련 완료! 네 안의 불꽃이 보여!"
```

### 3.3 읽기 이해 - "밀서 해독 훈련"
```yaml
상황: "혈귀의 정보가 담긴 밀서를 해독해야 해!"
등장 캐릭터: 카나오 (집중력 모델), 하루 (안내)
배경: 나비저택 비밀 서재
훈련 목표: 지문을 읽고 핵심 파악하기

시작 대사: "중요한 정보가 담긴 문서야!
          집중해서 읽고 내용을 파악해보자!"

완료 대사: "해독 완료! 네 집중력 정말 대단해!"
```

### 3.4 동기/정서 - "동료와의 대화 훈련"
```yaml
상황: "동료의 마음을 이해하는 것도 검사의 능력이야!"
등장 캐릭터: 젠이츠 (감정 표현), 탄지로 (공감)
배경: 나비저택 휴게실
훈련 목표: 감정과 동기 이해하기

시작 대사: "가끔은 싸우는 것보다 마음을 아는 게 더 중요해.
          네 마음은 지금 어떤 상태야?"

완료 대사: "마음 알기 완료! 너 자신을 더 잘 알게 됐어!"
```

### 3.5 문법 기초 - "시노부 선생님의 수업"
```yaml
상황: "정확한 말은 정확한 기술로 이어져!"
등장 캐릭터: 시노부 (선생님), 아오이 (조교)
배경: 나비저택 교실
훈련 목표: 문법과 맞춤법 익히기

시작 대사: "시노부 선생님 시간이야!
          올바른 표현을 익히면 호흡도 정확해져!"

완료 대사: "문법 훈련 완료! 시노부 선생님도 칭찬하실 거야!"
```

### 3.6 강점 발견 - "너만의 호흡 찾기"
```yaml
상황: "모든 검사에게는 특별한 재능이 있어!"
등장 캐릭터: 토키토 (영감), 이노스케 (도전)
배경: 나비저택 훈련장
훈련 목표: 자신만의 강점 발견

시작 대사: "너만의 특별한 힘이 있어!
          함께 찾아보자!"

완료 대사: "네 강점을 발견했어! 이게 바로 너만의 호흡이야!"
```

## 4. 캐릭터 활용

### 4.1 주요 등장 캐릭터

#### 하루 (주인공, 안내자)
- 모든 영역에서 동행
- 격려와 피드백 담당
- 레벨업 축하

#### 시노부 (지식 담당)
- 어휘력, 문법 영역
- 차분하고 친절한 설명
- 정답 시 미소

#### 렌고쿠 (동기 부여)
- 자기효능감 영역
- 열정적인 응원
- "마음을 불태워라!"

#### 탄지로 (공감)
- 동기/정서 영역
- 따뜻한 공감
- 함께하는 느낌

#### 토키토 (재능 발견)
- 강점 발견 영역
- 영감 제공
- 하루의 친구

### 4.2 캐릭터 대사 스타일

#### 시노부 스타일
```
"잘 하셨어요. 조금만 더 생각해볼까요?"
"대단해요. 금방 익히시네요."
"괜찮아요. 천천히 해도 돼요."
```

#### 렌고쿠 스타일
```
"훌륭하다! 마음의 불꽃이 타오르고 있어!"
"포기하지 마! 다시 도전이다!"
"대단해! 그 열정이야말로 검사의 자질이다!"
```

#### 탄지로 스타일
```
"잘했어! 정말 대단해!"
"괜찮아, 나도 처음엔 못했어."
"함께 하니까 든든해!"
```

## 5. 레벨별 스토리

### 5.1 레벨업 이벤트

#### 1형 → 2형 "첫걸음"
```
하루: "첫 번째 형을 익혔어!
      여우의 호흡 2형 - 월광질주(月光疾走)!
      이제 막 시작이야. 함께 더 강해지자!"
```

#### 2형 → 3형 "성장"
```
하루: "두 번째 형도 습득!
      여우의 호흡 3형 - 여우비(狐雨)!
      점점 실력이 늘고 있어!"
```

#### 3형 → 4형 "도약"
```
하루: "세 번째 형까지!
      여우의 호흡 4형 - 구미선풍(九尾旋風)!
      벌써 절반을 넘었어!"
```

#### 4형 → 5형 "각성"
```
하루: "네 번째 형!
      여우의 호흡 5형 - 월화연무(月華連舞)!
      대단해! 진짜 강해지고 있어!"
```

#### 5형 → 6형 "완성 직전"
```
하루: "다섯 번째 형!
      여우의 호흡 6형 - 여우화(狐火)!
      마지막 형이 보여!"
```

#### 6형 → 7형 "마스터"
```
하루: "드디어 완성!
      여우의 호흡 완전체 - 천구여우(天狗狐)!
      너는 이제 진정한 검사야!
      함께해서 정말 행복했어!"
```
